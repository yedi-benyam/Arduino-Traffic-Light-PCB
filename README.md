# Arduino Traffic Light PCB

## Overview
This project is a custom PCB designed in KiCad for an Arduinotraffic light circuit. 

## Features
- Three LED outputs (Red, Yellow, Green)
- Current-limiting resistors
- Compact PCB layout
- Designed using KiCad 10

## Components
- 3 × LEDs 
- 3 × 220 Ω Resistors
- Pin Headers
- Power and GND Connections

## Software Used
- KiCad 10

## Learning Objectives
This project was created to practice:
- Schematic design in KiCad
- PCB layout and routing
- Component placement
- Design rule verification (ERC & DRC)
- Preparing PCB designs for manufacturing
